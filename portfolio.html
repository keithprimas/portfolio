CacheCabinet- A MERN Stack Organizer App

"CacheCabinet" is a platform designed to enhance organization and productivity. Leveraging MongoDB, Express.js, React.js, and Node.js, users can create individual accounts with secure logins. Once logged in, users gain the ability to create, edit, save, and delete personalized collections of items tailored to their needs. This application not only empowers users to efficiently manage their lives but also showcases the versatility and effectiveness of the MERN stack in developing creative, user-friendly applications.

My tasks as a collaborator:
  • Fullstack Developer
  • GraphQL integration
  • App Design
  • Login and Register authentication logic
Challenges:

Establishing a connection from the client side to the MongoDB database presented challenges, given the asynchronous nature of JavaScript, necessitating precise handling of promises and callbacks. The integration of GraphQL mutations and queries introduced additional complexity, requiring thorough schema design and meticulous implementation of resolvers to facilitate smooth communication between the client and server, ensuring effective data flow and updates.

Technologies used:
  • React
  • GraphQL
  • Node.js
  • Express.js
  • MongoDB
  • Render
  • JWT
  • Bulma CSS
